# Multithreaded Web Server

## Overview
A Java-based web server built using Socket Programming and Multithreading concepts.

## Features
- Single Threaded Server
- Multi Threaded Server
- Client-Server Communication
- Socket Programming
- Concurrent Request Handling

## Technologies
- Java
- Socket API
- Threads
- IntelliJ IDEA
- Git & GitHub

## How to Run
1. Run Server.java
2. Run Client.java
3. Observe communication between client and server
